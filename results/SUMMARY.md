# TDA 실험 결과

노드 분류, test Macro-F1 (성능 주장이 아니라 실측치). 원본 metrics.json 은 `acm_ablation/`, `multidomain/` 에 있음.

## 1. 여러 도메인 (baseline=HAN 단독 vs full=GTN+PDGNN+HAN)

| 데이터셋 | 도메인 | baseline | full | Δ |
|----------|--------|----------|------|---|
| ACM | 학술/인용 | 0.9000 | 0.8961 | -0.0039 |
| DBLP | 학술/인용 | 0.7991 | 0.8737 | +0.0746 |
| FREEBASE | 지식그래프 | 0.1258 | 0.1286 | +0.0027 |
| IMDB | 영화 | — | — | (멀티라벨 → 현재 CE 파이프라인 미지원, 제외) |

- **DBLP 에서 위상이 크게 도움** (full +0.075). ACM 은 ≈(null). Freebase 는 노드특징 없음(one-hot)+subsample 이라 둘 다 학습 저조(macro-F1 0.13).
- 즉 **위상의 효용은 데이터셋마다 다르다** — node feature 가 약한 곳(DBLP)에서 위상이 크게 기여.
